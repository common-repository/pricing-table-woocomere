== Description ==

Pricing Tables plugin is to be designed for compatibility with Woocommerce and Wordpress to display pre-designed
pricing tables in a shortcode format using dynamic content from Woocommerce.

== Installation ==

1. Upload `pricing-woocomere-table` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
